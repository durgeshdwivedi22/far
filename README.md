# 🌾 Smart Farmer Assistant

A modern, production-ready web application for farmers with data tracking, crop suggestions, and analytics.

## ✨ Features

### 🌱 Core Features
- **Farmer Dashboard** - Complete overview with financial summaries
- **Crop Recommendation System** - AI-powered suggestions based on soil and season
- **Expense Tracker** - Track investments, profits, and losses
- **Analytics Page** - Beautiful charts and insights
- **Authentication** - Simple login/signup system

### 🎨 UI/UX Features
- **Modern Design** - Glassmorphism cards with gradient backgrounds
- **Dark/Light Mode** - Toggle between themes
- **Responsive Design** - Works perfectly on all devices
- **Smooth Animations** - Framer Motion powered transitions
- **Interactive Charts** - Recharts for data visualization

### 🧠 Smart Features
- **Auto Greeting** - Time-based welcome messages
- **Weather Widget** - Mock weather data with farming tips
- **Crop Tips** - Rotating farming advice
- **Data Persistence** - LocalStorage for data management

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ installed
- Modern web browser

### Installation

1. **Navigate to the project directory**
   ```bash
   cd smart-farmer-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   - Navigate to: `http://localhost:3000`
